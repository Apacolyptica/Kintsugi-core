 3  """ 
 4  Contains static XQuery strings for the individual CWE results pages.

 6  START:       Start of each CWE query

13  #All data-queries will start with an ID search

18  #This is a list of return values for the queries

22  """ 
23  These are query-ends that produce a single output string 
24  Enabling them to be concatenated 
25  """

33  These query-ends are about CWE relationships and generally produce more then one output each 
34  This prevents concatenation and the data must be processed differently 
35  NOTE: The database only shows relationshps 'going backward,' no 'ParentOf' relationships  
36  are shown from this query 
37  """
