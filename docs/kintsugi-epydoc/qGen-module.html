makeQuery(field, sType, fType, fList)
This method dynamically generates a BaseX query field is the field of weaknesses searched (example, 'Weakness') sType is the type of search being done (searching by ID, Name, etc.) fType is a LIST of filter categories to be appended to the query fList is a LIST OF LISTS of filter sub cattegories to be specified for each 'fType' example of filter syntax: ([Language, Operating_System_Type], [[C,C++,Java],[Windows_Vista]]) if fType and fList are blank, enter 0 (this will be updated to accept 'False' in the future)		 source code
